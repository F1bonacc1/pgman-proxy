## pgmctl topology

Render the cluster topology (peers, roles, sync standbys)

### Synopsis

Display the cluster topology in human-readable tree form by
default; -o json / yaml emit a schema-versioned document.

Derived from GET /v1/status — does not require any new server-side
endpoint.

```
pgmctl topology [flags]
```

### Options
